Emptying Log Interpreter

Welcome to the Emptying Log Interpreter web application. This tool allows you to visualize and analyze waste collection data with an interactive map interface.

Upload CSV File

Upload a local CSV file with emptying log data to visualize it on a map. The application supports various formats and data structures.

Supported File Format

The application supports CSV files with fields for timestamp, tag code, waste type, latitude, longitude, and weight. Various timestamp formats are automatically detected.

Interactive Maps

Visualize emptying locations with interactive maps and custom markers based on waste types.

Data Statistics

Get instant statistics about your emptying data, including waste type distribution.

Data Tables

View your data in structured tables for better analysis and understanding.
